# Sanitized Sprint Imports

Drop sanitized export files here from Tier 2 (restricted environment) sprints.

Naming convention: `{codename}-sprint-{N}.md`

These files contain ONLY process metrics and generalized observations -- no proprietary code, architecture, or business logic.

To incorporate into RESULTS.md, bring the import file to a Flowstate session and ask:
"Import this sanitized sprint export into RESULTS.md"
